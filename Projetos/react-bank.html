O que me levou a construir o projeto?

React Bank é um simulador de banco, onde busco testar o uso do Sass e treinar o desenvolvimento de interfaces, além de enfatizar o uso de React, Next js e seus benefícios.

Durante a criação da interface percebi que era possível implementar diversas funcionalidades e regras de negócio, todas a nivel de front-end.

Login

A primeira página do sistema é o login, onde devemos informar um usuário e senha para o sistema. Caso não informe você não consegue entrar no React Bank, todas as páginas possuem uma validação que verifica se você está logado com usuário e senha.

Saldo

O React Bank possui uma opção para você adicionar saldo na sua conta. O saldo serve para fazer transferências bancárias via pix ou da forma tradicional para amigos e fazer compras de produtos oferecidos pelo React Bank.

Lista de Amigos

Quando você seleciona a opção de pix ou transferência, você é levado para uma página com uma lista de amigos onde você pode enviar um valor desejado desde que ele não supere o seu saldo ou seu plano.

JSON

Primeiro precisamos construir um arquivo JSON com as informações de todos as pessoas que farão parte da lista de amigos, no meu caso utilizei esta estrutura.

Componente Friend

Agora devemos construir o componente Friend ele será responsável por fazer a leitura do nosso arquivo JSON, e preencher nosso article com suas informações. Através do uso do map no JSON podemos percorrer todos as pessoas e a medida que avançamos na leitura adicionamos as informações nas suas respectivas posições dentro de um article.

Página lista-de-amigos

Agora na página lista de amigos podemos importar nosso componente e adiciona-lo dentro de nossa section, onde todos os articles serão carregados automaticamente.
